# Demo assets

Static files for README and hackathon reviewers.

| File | Description |
|------|-------------|
| `sample_priority_hist.png` | Priority score distribution (generated by `scripts/run_pipeline.py`) |
| `demo_map_preview.png` | Optional screenshot of `outputs/map.html` |
| `architecture.png` | Optional exported architecture diagram |
| `demo.gif` | Optional 30–60s screen recording of the interactive map |

## Regenerate charts

```bash
python scripts/run_pipeline.py
```

## Add a demo GIF (manual)

1. Run the pipeline and open `outputs/map.html` in a browser.
2. Record pan/zoom and marker popups (OBS, Loom, or Win+G).
3. Save as `assets/demo.gif` and link from the root README.
